Overview

NEXUS Shop Manager is a private internal application used exclusively by the shop owner and authorized internal personnel to manage a single Etsy shop.

Purpose

This application streamlines day-to-day shop operations by connecting directly to Etsy and related manufacturing services to keep listings, inventory, orders, and fulfillment workflows in sync.

Core Functions

  • Listing Management: Create, update, and organize product listings with structured data and media.
  • Inventory Synchronization: Keep stock levels aligned between the shop and manufacturing systems.
  • Order Retrieval: Pull new and updated orders from Etsy into a single working view.
  • Fulfillment Workflows: Generate shipping labels, update tracking, and mark orders complete.

Access

NEXUS Shop Manager is not a public application. Access is restricted to the shop owner and explicitly authorized operators. No customer-facing features, public dashboards, or external onboarding flows are provided.

Important: This site does not offer a public sign-up or login flow. If you are a customer, please use the Etsy shop directly for purchases and support.
