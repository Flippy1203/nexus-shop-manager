[![License: Unlicense](https://img.shields.io/badge/license-Unlicense-blue.svg)](LICENSE)
[![Status](https://img.shields.io/badge/status-internal-green.svg)](#intended-use)

Private internal application for managing a single Etsy shop.

## Overview

NEXUS Shop Manager streamlines shop operations by connecting Etsy with internal manufacturing and fulfillment workflows. It is designed for use only by the shop owner and authorized internal personnel. The application is not a public service, and it is not intended for multi-tenant or multi-shop operation.

## Features

- **Listing management** — create, update, and organize product listings through the Etsy API.
- **Inventory synchronization** — keep stock levels aligned between the shop and manufacturing systems.
- **Order retrieval** — pull new and updated orders from Etsy into a single working view.
- **Fulfillment workflows** — generate shipping labels, update tracking, and mark orders complete.

## Intended Use

This is a private internal tool for a single shop owner. No public sign-up, customer login, or external onboarding is provided. Customers should use the Etsy shop directly for purchases and support.
