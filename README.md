# NEXUS Shop Manager

Private internal application for managing a single Etsy shop.

## Purpose

NEXUS Shop Manager streamlines shop operations by connecting Etsy with internal manufacturing and fulfillment workflows. It is designed for use only by the shop owner and authorized internal personnel.

## Core Functions

- **Listing Management** — create, update, and organize product listings
- **Inventory Synchronization** — keep stock levels aligned between the shop and manufacturing systems
- **Order Retrieval** — pull orders from Etsy into a single working view
- **Fulfillment Workflows** — generate labels, update tracking, and mark orders complete

## Important Notice

This is a private internal tool. No public sign-up, customer login, or external onboarding is provided. Customers should use the Etsy shop directly for purchases and support.
